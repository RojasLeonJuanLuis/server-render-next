# Server-Render-Next.js
> Made with Next.js in the next.js course

### Download repository

```
> git clone https://github.com/RojasLeonJuanLuis/server-render-next
```

### Run App
```
> npm install
> npm run dev
```

### Deploy
```
> npm run build
> now
> now alias [last-url].now.sh [new-url].now.sh
```
